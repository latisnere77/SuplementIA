# Product Overview

SuplementIA is an evidence-based health supplement recommendation platform powered by scientific research and AI.

## Core Features

- **Intelligent Search**: Vector-based semantic search with multilingual support (100+ languages)
- **Evidence Analysis**: Automated study counts and grades from PubMed
- **Personalized Recommendations**: Tailored for LATAM users with Spanish-first UX
- **Auto-Discovery**: Background workers automatically index new supplements from user searches
- **Subscription Model**: Monetization via Stripe with tiered access

## Target Users

- Health-conscious consumers in Latin America
- Spanish-speaking users seeking evidence-based supplement information
- Users looking for personalized supplement recommendations

## Key Differentiators

- Ultra-fast response times (< 200ms total, < 50ms edge cache)
- Scientific evidence backing (PubMed integration)
- Multilingual semantic search (not just keyword matching)
- Cost-optimized serverless architecture (~$25/month for 10K searches/day)
