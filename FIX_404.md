# Fix 404 Error - Chunks Not Loading

## Problema
Los chunks de webpack no se estaban cargando, causando error 404 en todos los recursos estáticos.

## Solución Aplicada

1. **Removido `generateBuildId`**: Esta configuración estaba causando problemas con la generación de chunks
2. **Simplificado `next.config.js`**: Removida configuración innecesaria que interfería con el build

## Cambios Realizados

### `next.config.js`
- ❌ Removido: `generateBuildId: async () => { return 'build-' + Date.now() }`
- ✅ Configuración limpia y estándar de Next.js

## Verificación

El build local ahora genera correctamente:
- ✅ Chunks en `.next/static/chunks/`
- ✅ Build exitoso sin errores
- ✅ Todos los recursos estáticos generados
