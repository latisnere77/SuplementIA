feat: Implement intelligent study search and ranking system

Implements a world-class intelligent search and ranking system for scientific studies
that provides objective, balanced, and high-quality evidence about supplements.

## Features

### Multi-Strategy Search (4 strategies combined)
- High Quality: RCT + Meta-analysis + Systematic Reviews
- Recent: Last 5 years
- Cochrane: Gold standard reviews
- Negative: Studies showing no effectiveness

### Advanced PubMed Integration
- History Server: Efficient multi-search combination
- Proximity Search: "magnesium glycinate"[Title:~3]
- Rate Limiting: 10 req/sec with API key
- Cochrane Priority: Maximum score (50 points)

### Multi-Dimensional Scoring (0-100)
- Methodology (0-50): Cochrane=50, Meta=40, RCT=30
- Recency (0-20): <2 years=20
- Sample Size (0-20): >1000=20
- Journal (0-5): Top tier=5
- Quality Tiers: 5 levels (exceptional ‚Üí low)

### Sentiment Analysis
- Claude Haiku: Classifies positive/negative/neutral
- Confidence Score: 0-1 with reasoning
- Batch Processing: Concurrency control
- Error Handling: Robust fallback

### Balanced Ranking
- Top 5 Positive: Best studies showing benefits
- Top 5 Negative: Best studies showing no benefits
- Consensus: 6 levels (strong_positive ‚Üí strong_negative)
- Confidence Score: 0-100

### Feature Flags
- USE_INTELLIGENT_SEARCH: Enable multi-strategy search
- USE_INTELLIGENT_RANKING: Enable sentiment + ranking
- Backward Compatible: Works with flags OFF
